# FindaLyric


### User Story

“Hold me closer, Tony Danza”
Wait, is that right?
Is there "a bathroom on the right", or "a bad moon on the rise?"
As a music aficionado, I want to find lyrics to my favorite songs so that I can tell that guy he has the wrong lyrics.
